# 📦 Installation de l'Extension Chrome Amazon Profit Pro

## Prérequis
- Google Chrome ou navigateur Chromium-based (Edge, Brave, etc.)
- Un compte Amazon (pour tester)

## Installation (5 minutes)

### Étape 1 : Télécharger l'extension
L'extension se trouve dans le dossier `extension/` de votre projet.

### Étape 2 : Ouvrir Chrome Extensions
1. Ouvrez Chrome
2. Tapez dans la barre d'adresse : `chrome://extensions/`
3. Ou cliquez sur les 3 points en haut à droite → Extensions → Gérer les extensions

### Étape 3 : Activer le mode développeur
En haut à droite de la page Extensions, activez l'interrupteur **"Mode développeur"**

### Étape 4 : Charger l'extension
1. Cliquez sur le bouton **"Charger l'extension non empaquetée"**
2. Naviguez jusqu'au dossier `extension/` de votre projet
3. Cliquez sur "Sélectionner le dossier"
4. ✅ L'extension est installée !

### Étape 5 : Épingler l'extension (optionnel mais recommandé)
Cliquez sur l'icône puzzle 🧩 dans la barre d'outils Chrome et épinglez l'extension Amazon Profit Pro.

## Utilisation

### Sur une page produit Amazon :
1. Allez sur n'importe quelle page produit Amazon
   Ex: https://www.amazon.fr/dp/B08N5WRWNW

2. Cliquez sur l'icône de l'extension 📦

3. Cliquez sur **"🔍 Extraire le produit Amazon"**

4. Les données sont automatiquement :
   - Sauvegardées dans le stockage local
   - Envoyées à l'application web (si ouverte)

### Dans l'application web :
1. Ouvrez https://trade-ai-global.vercel.app
2. Allez dans l'onglet **"🔄 Amazon→Alibaba"**
3. Les données extraites apparaissent automatiquement
4. Entrez le prix Alibaba trouvé
5. Voyez instantanément votre marge potentielle
6. Cliquez sur "💾 Sauvegarder" pour garder une trace

## Données extraites

✅ **Informations produit :**
- Titre complet
- Prix actuel (€/$/£)
- Images principales
- Note moyenne (⭐)
- Nombre d'avis clients
- ASIN (identifiant Amazon)
- URL du produit
- Caractéristiques techniques
- Dimensions et poids

## Dépannage

### L'extension ne fonctionne pas ?
1. Rechargez la page Amazon (F5)
2. Vérifiez que vous êtes bien sur une page produit Amazon
3. Réinstallez l'extension si nécessaire

### Les données n'apparaissent pas dans l'app web ?
1. Assurez-vous que l'app web est ouverte (rafraîchissez-la)
2. Vérifiez la console du navigateur (F12) pour les erreurs
3. Essayez de recharger l'extension
