# Changelog

Tous les changements notables de ce projet seront documentés dans ce fichier.

Le format est basé sur [Keep a Changelog](https://keepachangelog.com/fr/1.0.0/),
et ce projet adhère à [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [5.2.0] - 2026-06-30

### Ajouté
- Dashboard Analytics avancé avec statistiques globales
- Export CSV des analyses
- Graphique de distribution des marges
- Filtres et tri avancés
- Page témoignages utilisateurs
- Sélecteur de langue (FR/EN/DE/ES)
- Affichage de la version du logiciel
- ErrorBoundary pour la gestion d'erreurs
- Toast notifications
- ESLint + Prettier configurés
- Tests unitaires avec Jest
- Audit de code complet

### Changé
- Optimisation du bundle (79 kB)
- Amélioration des performances de chargement
- Correction des erreurs de syntaxe JSX
- Suppression des console.log en production

### Corrigé
- Erreur React #418 (hydratation)
- Problèmes de CSP
- Erreurs de préchargement de polices
