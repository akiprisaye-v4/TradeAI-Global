# 📸 Guide Screenshots Product Hunt

## Dimensions requises
- Format : 1270x760 pixels
- Ratio : 5:3
- Format : PNG ou JPG

## 5 Screenshots à créer

### Screenshot 1 : Dashboard Principal
- Montrer l'interface principale avec les onglets
- Mettre en avant le design sombre et orange
- Inclure quelques données d'exemple

### Screenshot 2 : Calcul de Marge
- Montrer le formulaire Amazon → Alibaba
- Afficher un exemple de calcul avec marge élevée
- Mettre en avant les chiffres clés

### Screenshot 3 : 10 Visualisations
- Montrer 2-3 visualisations différentes
- Graphiques colorés et professionnels
- Données réalistes

### Screenshot 4 : Export PDF
- Montrer le bouton d'export
- Aperçu d'un PDF généré
- Design professionnel

### Screenshot 5 : Mode Mobile PWA
- Capture d'écran mobile
- Montrer l'installation PWA
- Interface responsive

## Outils recommandés
- Figma (gratuit) : https://figma.com
- Canva : https://canva.com
- Capture d'écran : Snipping Tool (Windows) ou Cmd+Shift+4 (Mac)

## Astuces
- Utiliser des données réalistes mais fictives
- Mettre en avant les fonctionnalités uniques
- Garder le design cohérent avec l'app
