## 🚀 Démarrage Rapide

### Installation Locale

```bash
# Cloner le repo
git clone https://github.com/akiprisaye-v4/TradeAI-Global.git
cd TradeAI-Global

# Installer les dépendances
npm install

# Lancer en dev
npm run dev

# Build production
npm run build
# Installer Vercel CLI
npm i -g vercel

# Déployer
vercel --prod
