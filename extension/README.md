# Amazon Profit Pro - Chrome Extension

Extension Chrome gratuite pour extraire automatiquement les données des produits Amazon.

## Installation

1. Ouvrez Chrome et allez sur `chrome://extensions/`
2. Activez le "Mode développeur" (en haut à droite)
3. Cliquez sur "Charger l'extension non empaquetée"
4. Sélectionnez le dossier `extension` de ce projet
5. L'extension est installée !

## Utilisation

1. Allez sur une page produit Amazon (ex: amazon.fr/dp/B08N5WRWNW)
2. Cliquez sur l'icône de l'extension dans la barre d'outils
3. Cliquez sur "🔍 Extraire le produit Amazon"
4. Les données sont automatiquement envoyées à l'application web
5. Ouvrez https://trade-ai-global.vercel.app pour voir les données

## Fonctionnalités

✅ Extraction automatique :
- Titre du produit
- Prix actuel
- Images principales
- Note et nombre d'avis
- Caractéristiques techniques
- Dimensions et poids
- Catégorie

✅ Intégration avec Amazon Profit Pro :
- Les données sont synchronisées en temps réel
- Import direct dans le calculateur de profit
- Recherche automatique sur Alibaba

## Permissions requises

- `activeTab` : Pour accéder à la page Amazon active
- `storage` : Pour sauvegarder les données extraites
- `scripting` : Pour injecter le script d'extraction
- Accès aux domaines Amazon (.fr, .com, .de, etc.)
